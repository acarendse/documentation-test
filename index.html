POST /api/user

This service will be used to register a us number to be able to use the username provisioning plugin.

addUserSuccessRequestPayload

The request body to add a us number to use the plugin.

client_id

The unique Id that was supplied to the client, when the it registered with the plugin.

type
string
in
header

}
